Password Generator (Python)
About
This project generates a strong, random password based on the number of characters you choose.
Features
	•	Uppercase and lowercase letters
	•	Numbers
	•	Special characters
	•	Custom password length
How to Run
	1.	Install Python.
	2.	Run the file:
python password_generator.py
3.	Enter how many characters you want.
	4.	Your password will be generated.
File
	•	password_generator.py – main program file
Example
How many characters do you want in your password? 10
Your password is ready: A9@fLk3#Qp
